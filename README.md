💱 CurrencyPilot AI

A modern Currency Analytics Dashboard that provides real-time exchange rates, multi-currency conversion, interactive charts, and AI-powered market insights.

Built using HTML, CSS, JavaScript, and Chart.js, CurrencyPilot AI provides real-time currency analytics, interactive visualizations, and intelligent market insights through a modern fintech-inspired dashboard.

## 🎯 Key Highlights

- Supports 150+ global currencies
- Real-time exchange rate conversion
- Searchable currency dropdowns with flags
- Interactive exchange rate charts
- AI-powered market insights
- Multi-currency conversion
- Dark and Light themes
- Conversion history with local storage

✨ Features
💱 Real-Time Currency Conversion
Convert between global currencies instantly.
Live exchange rates fetched from an exchange-rate API.
Accurate and up-to-date conversion results.
⚡ Auto Convert
Automatic conversion while typing.
No need to repeatedly press the convert button.
Updates instantly when currencies are changed.
🌍 Multi-Currency Conversion

Convert a single amount into multiple currencies simultaneously.

Example:

INR
GBP
JPY
AED
BDT
And many more
🔍 Searchable Currency Dropdowns
Search currencies by code or name.
Currency flags for better visual recognition.
Fast and user-friendly selection experience.
📈 Exchange Rate Charts
Interactive exchange rate visualization.
Trend analysis using Chart.js.
Multiple time ranges supported.
🤖 AI Market Insights

Provides intelligent market summaries including:

Market sentiment
Best-performing currency
Weakest-performing currency
Suggested currency pair
🌓 Dark / Light Mode
Professional theme switching.
User preference saved using localStorage.
Smooth UI transitions.
🕒 Conversion History
Stores recent conversions.
History persists across page refreshes.
Clear history option available.
🎨 Modern Fintech UI
Glassmorphism design.
Responsive dashboard layout.
Inspired by modern fintech platforms such as Wise, Revolut, and XE.
