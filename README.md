# 💱 CurrencyPilot

### Navigate Global Exchange Rates with Ease

🚀 **Live Demo:** https://kaustubhdeshmane.github.io/CurrencyPilot/
A modern real-time currency converter supporting 30+ international currencies with live exchange rates.

---

## ✨ Features

- 🌎 Support for 30+ currencies
- 🔄 Currency swap functionality
- ⚡ Real-time exchange rates
- 🎨 Modern glassmorphism UI
- 📱 Responsive design
- 🚀 Fast and lightweight

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Exchange Rate API

---

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/KaustubhDeshmane/CurrencyPilot.git
```

2. Open index.html

3. Start converting currencies
